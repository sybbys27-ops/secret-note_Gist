### 🔐 1. 핵심 보안 구조 (밀리터리급 암호화)

이 앱의 가장 큰 특징은 **사용자의 브라우저 저장소(`localStorage`)에 원본 글(평문)을 절대 남기지 않는다**는 점입니다.

* **암호화 알고리즘 (AES-GCM 256-bit):** 현재 금융권과 보안 업계에서 표준으로 사용하는 가장 안전하고 빠른 암호화 방식을 사용했습니다.
* **키 생성 방식 (PBKDF2):** 사용자가 입력한 비밀번호를 그대로 쓰지 않고, 해시 함수를 150,000번 반복(Iterations)하여 해킹(무차별 대입 공격)을 방지하는 강력한 마스터 암호키로 변환합니다.
* **영지식(Zero-Knowledge) 증명 구조:** 마스터 비밀번호를 분실하면 제작자 본인조차도 절대 데이터를 복구할 수 없는 완벽한 철통 보안 구조를 가지고 있습니다.

### 📝 2. 주요 기능 및 UI 분석

다크 모드 기반의 세련된 UI(`--bg: #0f1115`)를 채택하여 해커의 터미널 같은 멋진 분위기를 풍깁니다.

* **잠금/해제 시스템:** 처음 접속 시 'LOCKED' 상태이며, 비밀번호를 입력해 성공적으로 복호화(해독)해야만 전체 UI가 활성화됩니다.
* **목록 관리:** 즐겨찾기(⭐ 핀 고정) 기능과 정렬(최근 수정순, 이름순) 기능을 제공합니다.
* **단축키 지원:** 개발자나 파워 유저를 위해 잠금 해제 상태에서 내용 작성 후 `Ctrl + S`를 누르면 즉시 암호화되어 저장되도록 설계되었습니다.
* **로컬 백업/복원:** [내보내기]를 하면 JSON 파일로 다운로드되는데, 이 파일 안의 데이터조차 **모두 암호문(Base64) 상태**이므로 USB에 담아 다녀도 매우 안전합니다.

### 💻 3. 코드 설계 요약

* **Crypto API 활용:** 자바스크립트의 내장 `crypto.subtle` API를 사용하여 외부 라이브러리 없이 순수 브라우저 기능만으로 고성능 암호화를 구현했습니다.
* **상태 관리 (State & Session):** `state` 객체에는 암호화된 파일만 담아서 로컬스토리지에 저장하고, 비밀번호와 해독된 데이터는 브라우저를 끄면 즉시 날아가는 휘발성 `session` 메모리에만 담아두어 보안을 극대화했습니다.
